# Jekyll Sitemap Generator Plugin

*Jekyll plugin to silently generate a sitemaps.org compliant sitemap for your Jekyll site*

[![Build Status](https://travis-ci.org/jekyll/jekyll-sitemap.svg?branch=master)](https://travis-ci.org/jekyll/jekyll-sitemap)

## Usage

1. Add `gem 'jekyll-sitemap'` to your site's Gemfile
2. Add the following to your site's `_config.yml`:

```yml
gems:
  - jekyll-sitemap
```

Make sure to place the `jekyll-sitemap` gem before any other content generating gem that you wish to exclude. As the sitemap will include all dirs and `.html` files.

## Developing locally

Use `script/bootstrap` to bootstrap your local development environment.

Use `script/console` to load a local IRB console with the Gem.

## Testing
